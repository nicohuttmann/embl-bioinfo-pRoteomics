8 Limma analysis

Here we are, after wrangling two days with our data, making it pretty and tidy without ever seeing a heatmap of it, ready to do a differential abundance analysis with limma. limma is a tool originally developed for microarray analysis, which works just as well for proteomics data.34 We will use it to identify, which peptides have a different abundance between the two sample groups.

8.1 Running limma

You can find the limma package on Bioconductor. Check out the website and find 1. how to install it, 2. the limma User’s Guide and 3. the Reference Manual.35 All Bioconductor package pages are made up this way and allow us to find the relevant information quickly. Except in this case.


# Install BiocManager if not yet installed
if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")
# Install limma
BiocManager::install("limma")

limma User’s Guide and Reference Manual


Admittedly, this is not very helpful yet. For other packages, you can find a much more concise documentation and commonly one or more workflow descriptions depending on what you want to do. This guide seems a bit more practical and can be applied to our data.

Let’s make this a real world example! Try and apply a limma analysis to your data. Use the explanations in this guide and on page 36/37 of the limma User’s Guide. Help for individual functions can be found in the Reference Manual and with the known ?limma::lmFit.

The final goal is to use the limma::topTable() function to extract the results. Have fun and good luck!
